# Outmoni Finance Upgrade Specification

## Authentication
- Added email/password login and signup alongside the existing Google authentication flow.
- Credentials auth registers or updates the user in the backend with a credentials provider identifier.
- Backend auth registration now supports Google and credentials users, optional password hashes, profile preferences, and provider metadata.

## Dashboard UI
- Rebuilt the dashboard into a modern dark command-center interface with glass panels, gradients, responsive KPI cards, and chart cards.
- Dashboard metrics include current balance, monthly income, monthly expenses, savings, net worth, spending this week, and upcoming bills.

## Transactions
- Transactions now model amount, type, category, note, merchant, payment method, account, date, tags, recurring flags, recurring frequency, and receipt attachment.
- UI supports create, update, delete, full-text search, pagination, and filters by type and category.
- Search covers category, amount, note, merchant, and tags.

## Categories
- Seeded income categories: salary, freelance, and business.
- Seeded expense categories: food, rent, shopping, and travel.
- Users can add custom income or expense categories from the dashboard.

## Accounts and Transfers
- Added multi-account dashboard cards for ICICI Bank, HDFC Bank, cash, UPI, credit card, and debit card.
- Transactions include transfer support for moving money between accounts.

## Budgets, Goals, Bills, and Notifications
- Added monthly, annual, and category budget visualizations.
- Added goals with saved amount, target amount, and progress percentage, including the buy-laptop goal.
- Added recurring transaction examples for salary, rent, and gym membership.
- Added upcoming bills with notification-style alerts for budgets, bills, recurring transactions, and goals.

## Analytics, Import, Export, and Profile
- Added charts for monthly spending, category-wise spending, cashflow, income vs expense, daily trend, and savings rate.
- Added dashboard actions for CSV bank statement import and CSV, Excel, PDF export.
- Added profile preference display for currency, timezone, dark mode, and language.
