# Outmoni Full-App Redesign Specification

## Authentication
- Google authentication remains available through NextAuth.
- Email/password signup now calls the backend register endpoint with `mode: register`.
- Email/password signin now calls the backend login endpoint with `mode: login`.
- Backend password storage now uses salted PBKDF2 hashes and login verifies passwords with timing-safe comparison.
- Backend auth returns a safe user payload instead of returning stored password hashes.

## Dashboard Application
- The dashboard has been redesigned as a modern dark finance command center.
- The dashboard is now local-first and persists transactions and custom categories in browser localStorage so changes survive refreshes.
- KPI cards show current balance, monthly income, monthly expenses, savings, net worth, and spending this week.

## Transactions
- Transactions support amount, type, category, note, merchant, payment method, account, date, tags, recurring flags, recurring frequency, and receipt attachment.
- Users can create, update, delete, search, paginate, and filter transactions.
- Search covers category, amount, note, merchant, and tags.
- Recurring transactions can be generated from existing recurring rows.

## Categories
- Built-in income categories include salary, freelance, and business.
- Built-in expense categories include food, rent, shopping, and travel.
- Users can create custom income or expense categories, and custom categories persist locally.

## Accounts and Transfers
- The account section includes ICICI Bank, HDFC Bank, cash, UPI, credit card, and debit card.
- Transfer transactions are supported through the `transfer` transaction type and account movement rows.

## Budgets, Goals, Bills, and Notifications
- Monthly and annual budget UI is available with category progress bars.
- Goals include saved amount, target amount, and progress percentage, including a buy-laptop goal.
- Upcoming bills show due dates, amount, account, and autopay/manual status.
- Notification panels warn about budgets, bills, recurring transactions, and goal milestones.

## Analytics
- Analytics charts include cashflow, daily trend, category-wise spending, monthly spending, and income-vs-expense.
- Savings rate is calculated from monthly income and monthly expenses.

## Import and Export
- CSV export downloads current transactions.
- JSON export provides an Excel-ready structured export for transactions, categories, accounts, bills, and goals.
- PDF export uses browser print so users can save the dashboard as PDF.
- CSV bank-statement import reads selected CSV files and adds imported rows to transactions.

## Profile Preferences
- Profile preference UI displays currency, timezone, dark mode, and language.
